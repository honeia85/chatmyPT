왜 ChatMyPT인가요?

프라이버시와 성능을 모두 잡은 로컬 AI 채팅 솔루션.

🔒

완전한 프라이버시

모든 대화가 로컬에서 처리됩니다. 데이터가 외부로 전송되지 않아 민감한 정보도 안심하고 대화할 수 있습니다.

⚡

빠른 응답 속도

경량화된 Qwen-1.8B 모델로 GPU 가속을 활용해 빠른 응답을 제공합니다. Apple Silicon과 CUDA 모두 지원합니다.

💬

자연스러운 대화

대화 히스토리를 기억하며 문맥을 이해합니다. 자연스럽고 연속적인 대화가 가능합니다.

🌐

웹 인터페이스

Gradio 기반의 깔끔한 웹 UI를 제공합니다. 브라우저만 있으면 어디서든 접속할 수 있습니다.

🧠

메모리 최적화

자동 메모리 관리로 장시간 사용에도 안정적입니다. 8GB RAM이면 충분합니다.

🔧

쉬운 설치

Python과 pip만 있으면 됩니다. 복잡한 설정 없이 몇 줄의 명령어로 바로 시작할 수 있습니다.

How it works

3단계로 시작하세요

복잡한 과정 없이 빠르게 설치하고 사용할 수 있습니다.

레포지토리 클론

GitHub에서 프로젝트를 클론하고 필요한 패키지를 설치합니다.

모델 다운로드

첫 실행 시 Qwen-1.8B 모델이 자동으로 다운로드됩니다.

대화 시작

브라우저에서 localhost:7860에 접속하면 바로 대화를 시작할 수 있습니다.
