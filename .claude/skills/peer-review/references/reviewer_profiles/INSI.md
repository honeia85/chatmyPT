# Insights into Imaging (INSI) — Reviewer Profile

## Submission System
- **Platform:** Editorial Manager (Springer Nature)
- **Review type:** Single-blind
- **Publisher:** ESR (European Society of Radiology) / Springer Nature

## Required Scorecard (4 items)

Rating scale: High / Medium / Low

1. Interest
2. Innovation
3. Importance
4. Language editing — categorical. Editorial Manager dropdown labels (verified 2026-05-16 on INSI-D-26-00015R1 confirmation PDF): "No (minor)" / "Yes (minor)" / "Yes (major)". The profile previously listed these as "No minor editing needed" / "Yes minor editing needed" / "Yes major editing needed"; the portal uses the parenthesized short form. Semantics: "No (minor)" = best/most favorable (well written, only minor polish possible); "Yes (minor)" = minor editing needed; "Yes (major)" = major editing needed.

## Additional Required Fields

- Transfer Authorization questions (Yes/Yes) — allow transfer to other Springer Nature journals if rejected.
- Conflict of interest declaration.

## Recommendation Options

- Accept
- Minor Revision
- Major Revision
- Reject (transfer to another journal allowed)
- Reject

## Confidential Framing — Minor Revision

When recommending Minor Revision, frame concerns as "labeling / clarificatory / reporting issue" vs "methodological flaw" and commit to "resolvable in a single revision cycle, no new analyses required". Helps the editor commit without protracted re-review.
