# chatmyPT

Qwen-1.8B 모델 기반 한국어 AI 시스템

## 특징

- Qwen-1.8B-Chat 모델 사용
- Apple Silicon(MPS) / CUDA / CPU 자동 감지
- 메모리 최적화
- Gradio 웹 인터페이스
- **멀티에이전트 오케스트레이터** - 대기업 조직 구조의 계층적 AI 에이전트 시스템

## 시스템 요구사항

- Python 3.8 이상
- 8GB 이상의 RAM
- CUDA 지원 GPU 또는 Apple Silicon

## 설치 방법

1. 저장소 클론
```bash
git clone https://github.com/honeia85/chatmyPT.git
cd chatmyPT
```

2. 가상환경 생성 및 활성화
```bash
conda create -n qwen_chat python=3.10
conda activate qwen_chat
```

3. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

## 실행 방법

### 기본 채팅
```bash
python app.py          # 웹 UI (http://localhost:7860)
python chat.py         # 터미널 CLI
```

### 멀티에이전트 오케스트레이터
```bash
python orchestrate.py  # 웹 UI (http://localhost:7861)
```

## 멀티에이전트 오케스트레이터

대기업 조직 구조를 모방한 계층적 AI 에이전트 시스템입니다.
`input/` 폴더에 파일을 넣고 대화하면, CEO가 적절한 부서에 업무를 배정합니다.

### 조직도

```
🏢 대표이사 (CEO) ─ 요청 분석, 업무 배정, 최종 종합
├─ 📊 데이터분석부
│  ├─ 📈 통계 분석가
│  └─ 🔍 패턴 분석가
├─ 📝 문서연구부
│  ├─ 📚 문헌 조사관
│  └─ ✂️ 요약 전문가
├─ 🔬 실험연구부
│  ├─ 🧪 가설 검증가
│  └─ ⚗️ 실험 설계가
└─ 📋 종합기획부
   ├─ 🖊️ 보고서 작성가
   └─ ✅ 품질 검토관
```

### 작동 방식

1. `input/` 폴더에 파일(PDF, CSV, TXT, JSON 등)을 넣습니다
2. 웹 UI에서 원하는 분석을 요청합니다
3. **CEO**가 요청을 분석하고 관련 부서에 업무를 배정합니다
4. **부서장**이 업무를 파악하고, 복잡하면 팀원에게 위임합니다
5. **팀원**이 세부 작업을 수행하고 부서장에게 보고합니다
6. CEO가 모든 부서의 결과를 종합하여 최종 답변을 제공합니다

### 지원 파일 형식

| 형식 | 확장자 | 처리 방식 |
|------|--------|----------|
| 텍스트 | .txt, .md, .log | 직접 읽기 |
| CSV | .csv, .tsv | 헤더 + 상위 20행 |
| PDF | .pdf | PyPDF2 텍스트 추출 |
| JSON | .json | 구조화된 텍스트 변환 |

### 설계 특징

- **동적 에이전트 생성**: 요청에 따라 필요한 부서/직원만 활성화
- **재귀적 위임**: 부서장 → 팀원으로 업무가 자동 세분화
- **단일 모델 공유**: 모든 에이전트가 하나의 모델을 공유하여 메모리 효율적
- **실시간 조직도**: UI에서 에이전트 트리와 진행 상태를 실시간 확인

## 주의사항

- 첫 실행 시 모델 다운로드에 시간이 소요될 수 있습니다
- Apple Silicon 사용 시 자동으로 MPS 가속을 활용합니다
- Qwen-1.8B의 8K 컨텍스트 윈도우 제약으로 파일 내용은 자동 요약됩니다
