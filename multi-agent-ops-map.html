현재 역할

  • 사용자와 직접 대화
  • 연구/트레이딩 총괄에게 백그라운드 작업 위임
  • 보드 상태와 보고서를 읽어 다시 브리핑

왜 이 페이지가 유용한가

  • 현재 어느 총괄이 어떤 worker를 굴리는지 한눈에 보임
  • 연구팀과 트레이딩팀 구조를 시각적으로 비교 가능
  • 나중에 실시간 상태판으로 확장하기 쉬움
