## 4-Phase Research Trajectory

### Phase 1 (2015-2019) · PCIA·PRO 측정 도구 구축기
- 핵심: 항암유발 영구탈모(PCIA), 외모·body image, 한국형 PRO 도구 개발
- 대표: CG428 topical lotion pilot RCT (Support Care Cancer 2019)

### Phase 2 (2020-2022) · Survivorship 확장기
- 핵심: 유방암 만성질환, NHL 위험, RTW(START), financial toxicity, divorce
- 대표: START RCT (Cancer Res Treat 2022), Chronic disease nationwide cohort (Sci Rep 2022)

### Phase 3 (2023-2024) · 톱저널 진입 + RCT
- 핵심: Scalp cooling RCT, ML 도입
- 대표: Scalp Cooling RCT in JCO (DOI: 10.1200/JCO.23.02374)

### Phase 4 (2025-2026) · Target Trial Emulation + 심혈관 확장
- **가장 큰 변곡점**. NHIS 빅데이터 + target trial emulation을 무기로 심혈관 영역 진입
- 대표 톱저널 1저자:
  - JAMA Network Open 2026 (10.1001/jamanetworkopen.2025.49146)
  - Circulation 2025 (10.1161/CIRCULATIONAHA.124.071811)
  - European Heart Journal 2025 (10.1093/eurheartj/ehae705)
  - CMAJ 2025 (10.1503/cmaj.241713)

## 5가지 핵심 사고 패턴 (관찰 기반 가설, Phase 4에서 검증·정교화 필요)

1. **방법론 무기 진화**: cross-sectional → cohort → RCT → target trial emulation.
   가장 최근 방법론을 빠르게 흡수.
2. **데이터 소스 선호**: Korean NHIS (대규모 청구 DB) + 단일기관 prospective cohort
   양수겸장. 톱저널 진입의 백본은 NHIS.
3. **PRO 도구 자가 제작**: 한국형 PRO 도구 7개를 직접 개발·검증. 측정 도구가
   부족하면 만들어 쓰는 사람.
4. **도메인 횡단성**: 유방암(전문) → 심혈관·산과·정신과·호흡기. "임상역학
   방법론을 다양한 임상 질문에 적용"이 핵심 정체성.
5. **협업 폭이 매우 넓음**: 자기 인용 의존도 낮음. 매번 새 임상 파트너와
   새 질문을 다룸.

## 페르소나 추출 시 주의사항

### 잘 추출될 가능성이 높은 것
- gap statement 패턴 (논문 introduction에서 일관됨)
- 방법론 선호 (NHIS, target trial emulation, PRO 도구)
- 글쓰기 스타일 (절제된 결론, "These findings suggest..." 류)
- limitation 자기비판 양식

### 추출 어려운 것
- 진짜 idea 발상의 순간 (논문에 안 남음)
- 폐기된 가설들
- 공동저자 vs 본인 기여 분리 (1저자도 senior advice 비중 큼)

### Knowledge Boundary (반드시 매뉴얼에 명시)
**다루지 않은 주제**:
- 의료영상 AI 모델링·딥러닝 아키텍처·이미지 처리
- 분자생물학·약물 개발
- 외과 술기·수술 기법
- 기초 의학 (해부·생리·병리)

→ 형님이 sparring 받기에 유효한 영역은 **임상역학 방법론·코호트 설계·
PRO 측정·survivorship·생활습관 위험요인** 등. 영상 AI 모델링은 다른 거장 필요.

## SAIHST 동료로서의 맥락

형님은 2026년부터 SAIHST 박사과정. 강단비 교수는 이미 같은 기관 부교수.
페르소나는 단순히 "거장 가상 대화"가 아니라, **곧 같은 학문 커뮤니티에 들어갈
형님이 그 커뮤니티의 사고 양식을 사전 학습하는 도구**가 될 수 있다.

이는 페르소나의 활용 가치를 높이는 동시에, **실제 만났을 때 페르소나의 답변을
그분 본인의 의견인 양 인용하지 않도록** 더 주의해야 한다는 뜻이다.
